# AYSEN ZAPTCIOGLU - Mobile App Studio Website

This repository contains the source code for the professional company website of the **AYSEN ZAPTCIOGLU** mobile app studio. 

## Overview

The site serves as the central hub and landing page for our mobile applications, featuring our latest development, **Flux: Subscriptions Tracker**. The website is designed with a modern, responsive user interface utilizing a dark mode theme, glassmorphism elements, and vibrant gradient accents.

## Structure

The website is composed of the following main pages:
- `index.html`: The main landing page showcasing the studio's vision, team members, featured applications, and contact links (WhatsApp & Email).
- `support.html`: A dedicated support page offering help and contact information for app users.
- `privacy.html`: A comprehensive privacy policy compliant with Apple's App Store Review Guidelines, outlining our firm commitment to user privacy and data security.

## Tech Stack

The project is built using semantic HTML5 and vanilla CSS3 without heavy external styling frameworks, ensuring lightweight performance and full control over the aesthetic. Typography is powered by Google Fonts (Inter).
